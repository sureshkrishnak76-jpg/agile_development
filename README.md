# Basic Python Logic and Operations

This repository contains basic Python scripts demonstrating core programming concepts. These include list operations, conditional structures, loops, and membership testing.

## Features Included

* **List Statistics**: Computes the sum, maximum, and average of a numerical list.
* **Weather Advisor**: Recommends clothing based on integer temperature thresholds.
* **Symptom Checker**: Evaluates basic health flags using logical `and` operators.
* **Rule-Based Chatbot**: Iterates through string messages to print corresponding bot replies.
* **Linear Search**: Uses the `in` keyword to verify if an element exists in a list.
* **Animal Classifier**: Identifies species based on structural features and sounds.

## Prerequisites

* Python 3.x

## How to Run

1. Clone or download this repository.
2. Open your terminal or command prompt.
3. Run the script using the following command:

```bash
python main.py
```
